# Matrix App - Вариант 4

Консольное приложение для работы с матрицами.

## Функции
- Ввод матрицы произвольной размерности
- Нахождение минимального элемента и его индексов
- Нахождение максимального элемента и его индексов
- Расчет статистики матрицы
- Интерактивное меню

## Использование
```bash
python matrix_app.py
